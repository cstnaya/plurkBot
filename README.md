PlurkBot
===

Make a plurkbot to follow someone's new SNS messages, and then poll plurks to notificate users.

## How to use

1. Prerequirement
   - Node.js

2. Steps
   - edit `config.json.example` file and rename it to `config.json`

   - rename `tz.json.example` to `tz.json`

3. run
   - `node src/index.js`
